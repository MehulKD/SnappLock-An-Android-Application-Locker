SnappLock Terminologies

Default Password: 1234
Please change the password from snapplock preferences!


Intruder: A person who tries to open the lock without the consent of the owner of the phone


Intruder log: Displays the picture of the intruder who tries to unlock the application but isn't successful


Intruder Shield Activated: SnappLock is enabled and it will protect the applications which are selected to be locked


FAQs

How to ENABLE SnappLock? Got to settings tab and switch 'ON' the Enable/Disable Switch and there will be service notification displayed on the


How to select applications to be LOCKED? : just click on the application name displayed in the list and there is a snapplock icon appearing on its right to show that application is locked. For example applications Facebook, Gallery & Play store are locked

lock all: lock all button will select all the applications to be locked.


lock important: this button will lock few important applications


unlock all: this button will deselect all the applications



How to remove applications from lock list? : just click on the application with the lock icon on right hand side the lock icon is gone and thus the application is removed from SnappLock Intruder Shield. This means that no password will be asked on start of that application.


What is SMS service? :when sms service is enabled a message is sent when wrong password is entered for three or more times to the number you enter in the preferences. Please enter the phone number and custom emergency message which you want to send on wrong password entry more than thrice!
